Установка на iPhone

Открой сайт в Safari, нажми Поделиться → На экран «Домой» → Добавить. После этого Srbingo будет открываться как отдельное приложение.

  1. Загрузи файлы в GitHub Pages.
  2. Открой опубликованный URL на iPhone в Safari.
  3. Добавь на главный экран — иконка уже подготовлена.

Озвучка работает через встроенные голоса браузера. На iOS первый запуск аудио обычно требует нажатия на кнопку 🔊.
