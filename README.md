# Srbingo — самоучитель сербского для GitHub Pages

Готовое статическое HTML-приложение для самостоятельного изучения сербского языка.

## Что внутри

- `index.html` — основной экран приложения.
- `styles.css` — адаптивный дизайн для ПК и iPhone.
- `app.js` — логика уроков, карточек, прогресса, озвучки, упражнений.
- `manifest.webmanifest` — PWA-настройки для установки на главный экран.
- `sw.js` — offline/cache service worker.
- `icons/` — иконки 192/512 px для iOS, Android и браузера.

## Возможности

- Уроки по темам: азбука, приветствия, базовые фразы, числа, кафе, город, дом, глаголы, первые падежи.
- Упражнения: выбор ответа, аудирование, набор слова, пары, сбор фразы, карточки.
- Произношение через Web Speech API (`sr-RS`), если голос поддерживается браузером.
- Переключение Serbian Latin / Serbian Cyrillic.
- Прогресс, XP, серия дней, сложные карточки — всё сохраняется в `localStorage`.
- Работает без базы данных, без внешних библиотек и без серверной части.

## Публикация на GitHub Pages

1. Распакуй архив.
2. Загрузи все файлы в репозиторий GitHub.
3. Включи GitHub Pages: `Settings` → `Pages` → `Deploy from branch` → выбери ветку `main` и папку `/root`.
4. Открой выданную ссылку.

## Установка на iPhone

1. Открой опубликованный сайт в Safari.
2. Нажми кнопку «Поделиться».
3. Выбери «На экран “Домой”».
4. Нажми «Добавить».

После этого приложение откроется как отдельная иконка.

## Как редактировать слова

Открой `app.js` и найди массив `vocab`. Формат карточки:

```js
w('zdravo','здраво','привет','greetings','greeting','Zdravo, kako si?','Привет, как ты?')
```

Поля: латиница, кириллица, русский перевод, урок, тег, пример на сербском, перевод примера.
